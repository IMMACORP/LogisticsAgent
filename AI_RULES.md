# AI Engineering Rules

## Stack
- Next.js App Router
- TypeScript strict mode
- Hono backend
- API Client Generator	Orval
- Prisma ORM
- PostgreSQL
- SDK: OpenAI Agents SDK
- LLM: GPT-4.1-mini

## Architecture
- Thin route handlers
- Business logic in services
- Shared schemas with zod
- Use repository pattern

## Naming
- camelCase variables
- PascalCase components
- snake_case DB columns

## UI
- shadcn/ui
- server components first
- TanStack Table

## AI Rules
- Never duplicate types
- Reuse existing schemas
- Check docs before generating code
